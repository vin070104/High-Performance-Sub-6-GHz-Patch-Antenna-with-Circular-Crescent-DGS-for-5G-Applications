# High-Performance-Sub-6-GHz-Patch-Antenna-with-Circular-Crescent-DGS-for-5G-Applications
Compact 3.5 GHz microstrip patch antenna using a circular-crescent DGS on FR4. The DGS improves S11 from –7.25 dB to –21.2 dB at 3.5 GHz and adds a –35.29 dB resonance at 3.28 GHz. Includes HFSS design files, parametric analysis, current plots, gain results and comparison with existing antennas.
